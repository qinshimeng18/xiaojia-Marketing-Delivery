# Market Kit Skills

这个仓库用于分发可安装的营销 AI skill。

当前包含：

- `skills/market-kit-skills`

它适合这些场景：

- campaign plan
- 小红书图文与笔记
- 人群、卖点、定位梳理
- 资料驱动的营销内容生成
- 营销图片和图文组合内容

## 安装方式

安装方式很简单：把这个 GitHub 仓库地址发给你的 Agent，然后告诉它帮你安装 `market-kit-skills` 就可以。

```text
请帮我安装这个 skill：
https://github.com/qinshimeng18/xiaojia-skills

skill 名称：
market-kit-skills
```

更具体的用法见：

- `skills/market-kit-skills/README.md`
- `skills/market-kit-skills/SKILL.md`
